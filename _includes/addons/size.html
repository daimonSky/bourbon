Size

View source

Set width and height in a single statement. Accepts all units, including auto and inherit, unitless numbers, and intrinsic keywords like fill, max-content, min-content & fit-content. You can also use this mixin with the calc() CSS function.

{% highlight scss %} @include size(2em); // width: 2em; height: 2em; @include size(10em auto); // width: 10em; height: auto; @include size(min-content 9rem); // width: min-content; height: 9rem; @include size(20px calc(100% - 80px)); // width: 20px; height: calc(100% - 80px); {% endhighlight %}
